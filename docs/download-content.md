## New features

- **Remote news feed** — announcement cards on the welcome screen, loaded from GitHub without a new app build.
- **News hub** — full articles on GitHub Pages with Markdown, linked from each card.
- **Show / hide news** — toggle the news row on the welcome screen when you want a cleaner view.
- **External model support** — connect providers and switch models from the composer.

## Improvements

- Smoother welcome-screen layout and spacing around the composer.
- News cards support background images, GIFs, and video wallpapers.
- Article pages render Markdown with headings, lists, tables, and code blocks.

## Notes

OpenChat for desktop is under active development. Some features in this feed are **experimental** and may change or be removed.

Release downloads will be linked here once a public build is available.
