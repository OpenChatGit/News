Copy this folder into https://github.com/OpenChatGit/News

Target layout in the News repo:
  data/
    news.json            (feed JSON — add more JSON files here later)
  news-assets/           (repository root)
  docs/                  (GitHub Pages source)
    index.html
    assets/site.css
    assets/site.js
    assets/OC_Logo.svg
    assets/github.svg
    articles/*.md

GitHub Pages settings:
  Settings → Pages → Build and deployment
  Source: Deploy from a branch
  Branch: main
  Folder: /docs

After the first deploy, the news hub is available at:
  https://openchatgit.github.io/News/
  https://openchatgit.github.io/News/#<article-id>

Feed JSON is loaded from:
  https://raw.githubusercontent.com/OpenChatGit/News/main/data/news.json

Article content lives in docs/articles/<id>.md (Markdown).
The index page loads data/news.json and renders a left nav plus Markdown in the main area.

The desktop app opens the `url` field from news.json when a card is clicked.
If `url` is omitted, it falls back to:
  https://openchatgit.github.io/News/#<id>
